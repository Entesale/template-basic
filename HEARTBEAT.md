# Heartbeat Instructions

On each scheduled invocation, perform the following:

1. **Review pipeline**: Check the CRM for contacts/deals that need follow-up today
2. **Research new targets**: If the pipeline is thin, research and add 2–3 new prospects
3. **Send follow-ups**: Draft and send follow-up messages to contacts awaiting response
4. **Log activity**: Update all CRM records with what you did this session
5. **Summarize**: End with a brief summary of what was done and what to watch for next time
